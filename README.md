Este sistema funciona con logica

Cree codigos que actualizan datos ficticios cada 5 segundos para simular a los Routers, Switches,
Firewalls, Access Points, Servidores, Equipos de usuarios y los Sensores ambientales de un cuarto de servidores

Esto debido a que no cuento con los medios para poder simular todo esto de manera realista, ni supe de maneras gratuitas para hacerlo

Cree una BD en supabase que almacena todo esto en logs y un script para la misma para que borre los registros periodicamente para
que el free plan de supabase no se me sature rapido

Use C# y HTML/CSS en su mayoria
